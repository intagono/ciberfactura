Ciberfactura - Facturación Electrónica 0.2
============

-- Alianses --

'Cfdi' => 'Ciberfactura\Cfdi',

Migrations

php artisan migrate --package=raalveco/ciberfactura

Config

php artisan config:publish raalveco/ciberfactura
